BAMily required input:

-L/-list


FILE


BAMily takes in a plain text file as input containting the path to each individual's sequence data (BAM format).
For each individual, a carrier status MUST be assigned (1 carrier, 0 non-carrier). For a nuclear family,
in which a father and a child are suspected of sharing a variant of interest not present in either the mother or the other child,
the file would be as follows:

/path/to/data/father.sorted.bam:1
/path/to/data/mother.sorted.bam:0
/path/to/data/child1.sorted.bam:1
/path/to/data/chil2.sort.bam:0

-R/-referenceFILEBAMily requires a reference sequence in the FASTA format (.fa or .fasta).

BAMily optional input:

-O/-ouputNAMEProvide a name for your output VCF file. ["results"]
-P/-priorDOUBLESet prior for the detection of a variant in a single individual. [0.001]
-Q/-minqualityDOUBLEProvide a minimum quality cutoff threshold. [0.2]
-I/-IndAllelesINTSet the number of independent alleles. [number of samples*2]

Example:

java -Xmx1024m -jar /path/to/BAMily.jar -L FamilyB.list -R Sapiens.HG19.fasta -O FamilyB
